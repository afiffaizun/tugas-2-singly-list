Buatlah program menggunakan bahasa pemrograman Java/Python yang mengimplementasikan struktur data Singly Linked List untuk menyimpan data mahasiswa dengan atribut:

-NIM (String)
-Nama (String)

Program harus memiliki:
Class Node untuk menyimpan data mahasiswa dan pointer ke node berikutnya. Class LinkedList untuk mengelola operasi pada linked list. Variabel head untuk menunjuk node pertama. Variabel count untuk menghitung jumlah data. Program harus menyediakan menu interaktif (menggunakan switch-case) dengan fitur:

Menu
-Insert at Beginning
-Insert at Given Position
-Insert at End
-Delete from Beginning
-Delete Given Position
-Delete from End
-Delete First Occurrence (by NIM)
-Show Data
-Exit

Ketentuan tambahan:
-Program harus melakukan validasi posisi agar tidak terjadi error.
-Program harus menangani kondisi list kosong.
-Output tampilan data harus rapi dalam bentuk tabel (menggunakan System.out.printf).
-Sertakan komentar penjelasan sederhana pada setiap bagian penting kode.
-Gunakan Scanner untuk input dari user.

Buat program Java berbasis OOP yang terdiri dari class Node, class LinkedList, dan class Main. Implementasikan Singly Linked List dengan atribut NIM dan Nama. Sediakan seluruh operasi insert dan delete (awal, posisi tertentu, akhir, dan berdasarkan NIM). Tambahkan menu interaktif menggunakan switch-case, validasi input, pengecekan list kosong, dan tampilkan data dalam format tabel rapi. Gunakan Scanner untuk input dan tambahkan komentar sederhana pada setiap method.
